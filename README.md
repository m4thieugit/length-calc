# 📦 length-calc

**length-calc** est un outil permettant de *calpiner* (optimiser la disposition) de différents formats sur une laize définie ou sur une plaque.

Il aide à maximiser l’utilisation de matière en organisant efficacement les dimensions données.

---

## 🚀 Fonctionnalités

- 📏 Calpinage sur **laize (largeur fixe)**
- 🧱 Calpinage sur **plaque (surface définie)**
- 📐 Gestion de **formats multiples**
- ⚡ Optimisation de la disposition pour limiter les pertes

---

## 🧠 Principe

L’outil prend en entrée plusieurs formats (dimensions) et les organise automatiquement :

- soit sur une **laize donnée**
- soit sur une **plaque complète**

afin de :

- réduire les chutes
- améliorer le rendement matière
- faciliter la préparation de production

---

## 🛠️ Utilisation

1. Définir la **laize** ou les dimensions de la **plaque**
2. Ajouter les formats à calpiner
3. Lancer le calcul
4. Obtenir une disposition optimisée

---

## 📌 Exemple d’usage

- Industrie textile (optimisation de coupe)
- Impression grand format
- Découpe de matériaux (bois, métal, plastique)
